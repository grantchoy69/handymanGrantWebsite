# Captioned Before/After Images

These WebP files use the same filenames as the optimized before/after images
already referenced by the Handyman Grant website.

Each image now includes:

- Medium-gray outer border
- Gray BEFORE and AFTER header bars
- Center divider
- White high-contrast labels
- Website-friendly WebP compression

Copy the files into the repository's `photos/` folder and replace the existing
files with the same names.
