# Handyman Grant Work Gallery

Replace the repository's `work.html` with the included file.

Also copy the twelve WebP files in `photos/` into the repository's existing
`photos/` folder. They use the same filenames as the captioned before/after
images created previously.

The page references additional finished-project images already used elsewhere
in the current Handyman Grant repository.

Features:

- Twelve captioned before-and-after project cards
- Curated finished welding, door, drywall, lighting, mounting, and custom work
- Caption beneath every image
- Responsive card grid
- Lazy-loaded images
- Keyboard-accessible click-to-enlarge modal
- Handyman Grant logo and current site styling
- Existing Google Ads call-conversion tracking
- Floating Call Grant button
